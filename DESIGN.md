# Design

## Visual Theme
Light mode with a "Paper-white" aesthetic. Tinted neutrals using OKLCH.

## Color Palette
Restrained but committed strategy.
- **Primary**: OKLCH(20% 0.01 222) - Deep Slate
- **Accent**: OKLCH(60% 0.15 222) - Sky Blue
- **Background**: OKLCH(98% 0.005 40) - Off-white

## Typography
- **Primary**: Sarabun (for Thai clarity)
- **Secondary**: Inter (for UI elements)
- **Scale**: 1.25+ ratio. Heading 1 (7xl+), Body (lg).

## Spacing & Rhythm
Loose spacing with varying rhythms. Avoid uniform padding. Use 65-75ch for text blocks.

## Components
- **Cards**: Large radius (3rem+), subtle shadows, no borders.
- **Buttons**: Rounded-2xl, high contrast.
- **Icons**: Custom or carefully selected Lucide icons.

## Motion
Purposeful ease-out-quart curves. No bounce.
